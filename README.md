# superstore-sales-analysis
End-to-end sales analysis using Advanced Excel — KPIs, Pivot Tables and Interactive Dashboard.
## Business Problem
Analyzed Superstore sales data to identify
top performing products, regional gaps,
and profitability trends.

## Tools Used
Advanced Excel — Pivot Tables, KPIs,
Interactive Dashboard, VLOOKUP, INDEX-MATCH

## Key Features
- KPI cards: Total Sales, Total Profit,
  Profit Margin, Average Order Value
- Regional sales breakdown
- Product category performance analysis
- Monthly trend analysis
- Interactive dashboard with slicers
